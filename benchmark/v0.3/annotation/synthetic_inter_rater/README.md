# Synthetic inter-rater tier

Agreement statistics that use a deterministic jitter layer for rater B live
under repository root `annotation/rater_a.csv` and `annotation/rater_b.csv`
(generated by `scripts/materialize_v03_adjudication_artifacts.py`).

If an independent second human pass is available, place
`annotation/rater_b_human.csv` and rerun
`python scripts/reproduce_agreement_report.py`; the wrapper prefers
`rater_b_human.csv` automatically.

See [docs/PROVENANCE.md](../../../../docs/PROVENANCE.md).
